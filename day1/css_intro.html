INTRODUCTION ABOUT CSS

CSS

Cascading style sheet
Using CSS we can create RWD (Responsive Web Design)
Responsive means reaction
Web design means we display our content based on our desktop size

CSS stands for Cascading Style Sheets

CSS describes how HTML elements are to be displayed on screen, paper, or in other media
CSS saves a lot of work. It can control the layout of multiple web pages all at once
External stylesheets are stored in CSS files

Why do we need CSS

  • Without CSS, all web pages would look plain and boring. CSS allows us to:
  • Change colors, fonts, sizes
  • Add margins, padding, borders
  • Make the page Responsive (fit different screen sizes)
  • Create animations and transitions
  • Build modern layouts like grids and flexboxes

Three Ways to Insert CSS into HTML

CSS can be added to HTML in three different ways:

  1. Inline CSS

    2. CSS is added directly inside an HTML tag using the style attribute

    • Syntax:

      • <tag style="property: value;">Content</tag>

      Example :

      <p style="color:red;font-size:20px;">This is red text</p>

      Use When:
    • You need to style one element only
    • For quick testing or debugging

  2. Internal CSS

    3. CSS is written inside the <style> tag in the <head> section of the HTML file

    • Syntax:
      • <head>
                <style>
                    selector {
                        property: value;
                    }
                </style>
                </head>
      Example:
      <!DOCTYPE html>
    <html>
        <head>
             <style>
                p {
                    color: blue;
                    font-size: 18px;
                    }
                </style>
        </head>
         <body>
            <p>This is a blue paragraph.</p>
        </body>
        </html>
      Use When:
    • You are working on a single HTML file
    • Small projects or quick demos

  3. External CSS

    4. CSS is written in a separate file with a .css extension, and it is linked to the HTML file using the <link> tag.

    • Syntax to Link:

      • <link rel="stylesheet" href="style.css">
      Example:

      style.css

      h1 {
    color: green;
    text-align: center;
        }

      index.html

      <!DOCTYPE html>
    <html>
        <head>
            <link rel="stylesheet" href="style.css">
        </head>
        <body>
            <h1>Welcome to CSS!</h1>
        </body>
        </html>
      Use When:
    • You are working on large or multiple-page websites
    • You want to reuse styles across many pages
    • Best practice for clean code and separation of concerns
